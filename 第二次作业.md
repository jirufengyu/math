<h2 id="递归">递归</h2>
<ul>
<li>一个函数重复调用自己</li>
<li>有终止条件</li>
</ul>
<blockquote>
<p>n个台阶，可以一次走1步，也可以一次走2步，有多少种走法？</p>
</blockquote>
<p>f(n)=f(n-1)+f(n-2)</p>
<pre><code>int f(int n){
    if (n==1) return 1;
    if (n==2) return 2;
    return f(n-1)+f(n-2);
}
</code></pre>
